----------------------------------------------------------------------------------
help: mostra os comandos e uma ajuda

dir: Mostra s diretorios e arquivos da pasta inicial

cd: abre uma pasta ou arquivo
	para voltar para trás em uma pasta é 
		'''
		cd ..
		'''

	para ir para a raiz do computador: 
		'''
		cd /
		'''

tree: mostra todas as arvores de arquivos do diretório atual.
	para mostrar aos poucos:
		'''
		tree | more
		'''

color: muda as cores do cmd
	para ver todas as opções de cores:
		'''
		color /?
		'''
	para deixar verde (hacker):
		'''
		color 2
		'''

mkdir: cria uma pasta no diretório atual.

rmdir: remove a pasta especifica no diretório atual

shutdown /S /F /T 120: desliga o PC em 120s
	/S: desligar
	/F: forçar o desligamento
	/T: tempo

shutdown /A: Cancela o desligamento do computador

start: inicia o cmd

start notepad: inicia o programa escrito

FOR /L %A in (1,1,10)do mkdir pasta%A: Esse código faz um laço for na pasta atual, criando 200 pastas.
	o %A é o nome da variável (seria equivalente ao i na programação).
	após o do, é tudo que vai acontecer no escopo do for.
	criamos pastas com o mkdir chamadas pasta+i

FOR /L %A in (1,1,200)do rmdir pasta%A: remove as 200 pastas
